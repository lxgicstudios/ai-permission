AI-powered RBAC generator. Creates role-based access control configs from your API routes automatically.

Generate RBAC permission configs from your routes using AI.

## Install

```bash
npm install -g ai-permission
```

## Usage

```bash
npx ai-permission ./src/routes/
npx ai-permission ./api/
```

## Setup

```bash
export OPENAI_API_KEY=sk-...
```
